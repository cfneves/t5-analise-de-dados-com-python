1. Arquivos: CSV e JSON

Até agora, todo dado que você usou existia só enquanto o programa rodava. Arquivos permitem guardar e recuperar dados entre uma execução e outra.

FormatoUso comumMódulo Python
CSVPlanilhas simples, exportação de sistemascsv
JSONConfigurações, dados de APIs webjson
ExcelPlanilhas com formatação e múltiplas abasopenpyxl (ou Pandas, a partir do Bloco 3)

Exemplo 1 (resolvido) — Escrevendo um arquivo JSON

import json

produto = {"nome": "Mouse", "preco": 45.0, "estoque": 12}

with open("produto.json", "w", encoding="utf-8") as arquivo:
    json.dump(produto, arquivo, ensure_ascii=False)

print("Arquivo salvo com sucesso!")

Exemplo 2 (resolvido) — Lendo um arquivo JSON

import json

with open("produto.json", "r", encoding="utf-8") as arquivo:
    dados = json.load(arquivo)

print(f"{dados['nome']} custa R$ {dados['preco']}")
✏️ Agora é sua vez: no notebook, escreva uma lista de dicionários (alunos) em um arquivo CSV e leia esse arquivo de volta.

2. Datas e Expressões Regulares

Datas com datetime

from datetime import date

hoje = date.today()
fim_do_curso = date(2026, 8, 14)

dias_restantes = (fim_do_curso - hoje).days
print(f"Faltam {dias_restantes} dias para o fim do curso.")

Expressões regulares com re

Uma expressão regular (regex) é um padrão de texto usado para validar ou buscar formatos específicos — como e-mails, telefones ou CPFs.

import re

email = "aluno@exemplo.com"
padrao = r".+@.+\..+"

if re.match(padrao, email):
    print("E-mail parece válido!")
else:
    print("E-mail inválido.")
💡 Você não precisa decorar regex de memória — no mercado, a maioria dos profissionais consulta ou usa ferramentas para gerar o padrão certo. O importante é entender o que esse padrão verifica.

3. Funções: parâmetros, valores padrão e return

def calcular_total(preco, quantidade, desconto=0):
    return preco * quantidade - desconto

print(calcular_total(10, 2))        # sem desconto: 20
print(calcular_total(10, 2, 5))     # com desconto: 15
💡 "Sem return, a função é surda — não responde nada." Uma função sem return pode até imprimir algo na tela, mas não devolve nenhum valor que você possa guardar em uma variável ou usar em outro cálculo.

Exemplo 1 (resolvido) — Função com retorno

def media(lista):
    return sum(lista) / len(lista)

notas = [7.5, 8.0, 6.5]
print(media(notas))

Exemplo 2 (resolvido) — Parâmetro opcional

def saudacao(nome, formal=False):
    if formal:
        return f"Bom dia, {nome}."
    return f"Oi, {nome}!"

print(saudacao("Ana"))
print(saudacao("Ana", formal=True))
✏️ Agora é sua vez: no notebook, crie a função calcular_total e teste com e sem desconto.

4. Lambda e Módulos

Função lambda (função de uma linha)

media = lambda lista: sum(lista) / len(lista)

print(media([7.5, 8.0, 6.5]))

Uma lambda é equivalente a uma função criada com def, só que escrita em uma única linha — útil para funções pequenas e simples.

Criando seu próprio módulo

Um módulo é só um arquivo .py com funções, que pode ser importado em qualquer outro script — assim você nunca precisa reescrever a mesma função duas vezes.

# arquivo: utilidades.py
def media(lista):
    return sum(lista) / len(lista)

def calcular_total(preco, quantidade, desconto=0):
    return preco * quantidade - desconto
# outro arquivo, na mesma pasta
import utilidades

print(utilidades.media([7.5, 8.0, 6.5]))
print(utilidades.calcular_total(10, 2, 5))
✏️ Agora é sua vez: crie seu próprio utilidades.py com pelo menos 3 funções e importe-o em outro script.
